# Manual práctico personalizado – Serverless Inteligente

Este manual contiene las instrucciones paso a paso para ejecutar las tareas prácticas del proyecto Serverless Inteligente.
Cada lección se identifica con un código de bloque: L{número de lección}-S{sección}-B{bloque}.

Estructura:
- Lección 1: Introducción a la arquitectura sin servidor
- Lección 2: Sitio estático serverless
- Lección 3: Funciones como servicio (FaaS)
- Lección 4: API Gateway
- Lección 5: Persistencia serverless
- Lección 6: Representación en Cloudcraft
- Lección 7: Crecimiento y optimización

Cada bloque contendrá:
1. Objetivo del bloque.
2. Instrucciones paso a paso (AWS Console y/o CLI).
3. Resultado esperado.
4. Identificador del bloque (para retomar el hilo si ocurre un error).
